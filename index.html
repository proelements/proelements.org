What is included?

  • Automatic updates
  • All PRO widgets

  • Marketing tools integration
  • Full site editor
  • Global widget
  • Motion effects
  • Scrolling and mouse effects
  • Global Custom CSS
  • Custom fonts with TypeKit integration
